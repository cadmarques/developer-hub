TaskManager API

Uma API REST para gestão de tarefas, construída com boas práticas de engenharia de software.

Tecnologias e Princípios

Desenvolvido com foco em manutenibilidade, testabilidade e clareza arquitetural.

Java 17 Spring Boot 3 SOLID JUnit 5 Mockito Swagger / OpenAPI REST API GlobalExceptionHandler

Funcionalidades Principais

  • Criar, listar, atualizar e remover tarefas
  • Validação rigorosa de entrada (ex: título não vazio)
  • Tratamento global de exceções com `@ControllerAdvice`
  • Documentação interativa via Swagger UI (`/swagger-ui.html`)
  • Testes automatizados com JUnit 5 e Mockito para serviços e controladores

Tratamento Global de Exceções

Diagrama do fluxo de exceções não tratadas, capturadas pelo GlobalExceptionHandler.
